# **App Name**: GTNPlay

## Core Features:

- Video Catalog: Display a curated catalog of tv channels with titles and descriptions, mirroring the video tiles shown in image_0.png.
- Left Navigation: Replicate the left navigation bar featuring the GTNPlay logo, search, home, and menu icons.
- Hero Section: Faithfully reproduce the hero section with the "Unlock a World of Entertainment with Webkul" heading and descriptive text from image_0.png.
- Search Functionality: Enable users to search for videos by title or description using the search icon in the navigation bar.
- Video Playback: Allow users to select and play tv channels from the catalog.

## Style Guidelines:

- Primary color: Dark blue (#0A2540), emulating the ocean pier background, creating a sophisticated and immersive viewing environment.
- Background color: Very dark blue (#071A2B), closely matching the moody pier background of the reference image, and almost but not quite the same as the primary color.
- Accent color: Light sky blue (#7AD1FF) used for icons, navigation highlights, and interactive elements to provide a contrast and guide user focus in the interface.
- Font pairing: 'Inter' (sans-serif) for the headings and main text, and 'PT Sans' (sans-serif) for subtitles and smaller UI elements.
- Faithfully reproduce the layout from image_0.png, including the left navigation, hero section, and video tiles.
- Use icons that closely resemble the icons in image_0.png, including the GTNPlay logo, search icon, home icon, and menu icon.
- Implement smooth transitions and subtle animations when navigating between sections or selecting videos to enhance the user experience.
